Step 3 — Add the script to your Google Sheet:
1. Open the Google Sheet that JotForm syncs to
2. Click Extensions → Apps Script
3. Delete everything in the editor, paste the contents of Google_Apps_Script_Proxy.js
4. Click Save (💾), then Deploy → New deployment
5. Type: Web app · Execute as: Me · Access: Anyone
6. Click Deploy → Authorize → Allow
7. Copy the Web app URL (starts with script.google.com/macros/…)
